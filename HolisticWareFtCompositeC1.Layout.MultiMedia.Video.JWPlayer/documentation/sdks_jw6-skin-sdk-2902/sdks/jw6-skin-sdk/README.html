JW6 Skin SDK

Thanks for downloading the JW6 Skin SDK! This download to help you build your own skins for JW Player version 6. Inside, you will find two subdirectories:

skins
This directory contains the default skins of both version 5 and 6 of JW Player. They can be used as an example implementation or as a starting point for your own skin.
tests
This directory contains a test page with three JW6 setups to test your skin. Note that, in order for this page to work (in Flash), you have to load it from a webserver (or localhost)!

Note the Building JW Player Skins and Skin XML Reference documents can be found on our support site.

Good luck building your JW Player skins! As always, feedback is welcome and help is provided on our support forum.
